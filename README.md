<h1 align="center">aso-snack-shop</h1>

## 🔗 Public Link
http://jousen5.aso-abcc.com/sweeter/ (予定)

## 🔌 Usage
`WEB-INF/lib/`の中に[ここからダウンロード](https://school.yanai-k.com/aso-snack-shop_javalibs_v3.zip)したZipファイルの中にあるjarファイルを設置します。

## 💡 Features
* 管理画面
    * 商品 追加/編集/削除
* お気に入り機能
